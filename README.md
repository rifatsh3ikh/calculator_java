# Java Calculator

A simple calculator application written in Java.

## Description

This project is a basic calculator built using the Java programming language. It performs fundamental arithmetic operations and is intended for learning and practice purposes.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Simple and easy-to-understand code structure

## Technologies Used
